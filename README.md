# API Testing with CI/CD

![GitHub Actions Status](https://github.com/ваш-username/api-testing-ci-cd/actions/workflows/run-tests.yml/badge.svg)

Проект для автоматического тестирования REST API с использованием Python и GitHub Actions.

## Описание

Этот проект содержит:
- Автоматические тесты для JSONPlaceholder API
- Конфигурацию CI/CD с GitHub Actions
- Отчеты о тестировании и покрытии кода

## Технологии

- Python 3.9+
- pytest
- requests
- GitHub Actions
